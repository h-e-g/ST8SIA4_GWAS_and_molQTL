# ST8SIA4_GWAS_and_molQTL
Code for mapping molecular QTLs of ST8SIA4 and their colocalization with GWAS traits (Boyance et al,2026)


*data* folder contains the data structure expected to run the scripts.
Large data files such as RNA-seq fastqs/bam and genotypes from (Quach et al, 2016), GWAS summary statistics, credible set and colocalization results, are not provided and should be obtained from EGA, GWAS catalog or ope,n targets platform prior to running the code. 

*final_figures* folder contains the expected output from the scripts including figures 1A-D, and associated source data.
